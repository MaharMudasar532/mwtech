# MW Technical Services Provider Homepage

Modern, responsive corporate homepage built with React + Vite + Tailwind CSS.

## Tech Stack

- React 19
- TypeScript (app entry)
- Tailwind CSS 3
- Vite 8

## Theme Colors

Custom reusable brand colors are defined in [theme/colors.js](theme/colors.js) and consumed in [tailwind.config.js](tailwind.config.js):

- `mw.primary` → `#162C44`
- `mw.secondary` → `#7EA1C1`
- `mw.accent` → `#4F7FA8`
- `mw.background` → `#FFFFFF`

## Sections Implemented

- HeroSection
- ServicesSection
- WhyChooseUs
- WorkProcess
- CompletedWork / Portfolio
- Testimonials
- EmergencyBanner
- ContactSection
- Footer

## Run Locally

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
npm run preview
```
